# 🐭 Freedom Mouse

A chaotic mouse wiggler that keeps your computer awake and your status active. Perfect for those "busy" moments!

## 🌟 Features

- 🎨 Clean dark mode interface
- 🎯 Multiple movement patterns:
  - Random (chaos mode)
  - Circle (round and round we go)
  - Figure 8 (for the fancy folks)
  - Square (because geometry)
- ⚡ Customizable movement range
- ⏰ Adjustable intervals
- 🎲 Random interval option
- 💤 System sleep prevention
- 🛟 Emergency escape with ESC key
- 📊 Activity statistics tracking

## 🚀 Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/freedom-mouse.git
cd freedom-mouse
```

2. Install the requirements:
```bash
pip install -r requirements.txt
```

3. Run the program:
```bash
python mouse_wiggler.py
```

## 🎮 Usage

1. Select your preferred movement pattern
2. Adjust the movement range (10-200 pixels)
3. Set the interval between moves (1-60 seconds)
4. Optional: Enable random intervals for extra stealth
5. Click "Start Freedom" and watch the magic happen
6. Press ESC anytime for emergency stop

## ⚙️ Requirements

- Python 3.6+
- PyQt5
- PyAutoGUI
- keyboard

## 🛡️ Safety Features

- ESC key emergency stop
- Mouse failsafe (move to screen corner to stop)
- Screen boundary protection
